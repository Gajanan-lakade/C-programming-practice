# Basics of C Programming

This repository is created for learning and practice purposes.
Programs are based on standard C examples, rewritten and organized to understand core concepts.

## Programs Included
- `hello_world.c` – Prints Hello World with the help of printf() predefined function
- `data_types.c` – Demonstrates basic data types in C
- `operators.c` – Demonstrates arithmetic, relational and logical operators
- `scanf_example.c` – Demonstrates the scanf() pre defined function
`
## Key Concepts Covered
- Input using scanf() and output using printf()
- Basic data types (int, float, char, double)
- Arithmetic, relational, and logical operators
- Using format specifiers (%d, %f, %c, etc.)
