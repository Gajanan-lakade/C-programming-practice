# Conditional Statements in C Programming

This folder contains C programs created for **learning and practice purposes**.
The programs are based on standard C programming examples and are organized to
understand how decision-making works in C.

## Programs Included

- `if_else.c`  
  Demonstrates the use of **if-else** to check whether a number is even or odd.

- `else_if.c`  
  Demonstrates an **else-if ladder** to calculate grades based on marks.

- `switch_case.c`  
  Demonstrates the use of **switch-case** through a simple calculator example.

## Key Concepts Covered
- Decision making using conditions
- Logical expressions
- If-else control flow
- Else-if ladder
- Switch-case statement
