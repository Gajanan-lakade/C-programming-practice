# Functions in C Programming

This folder contains programs demonstrating how functions work in C.
Functions help in modular programming by dividing large programs into
smaller, reusable blocks of code.

## Programs Included
- `basic_function.c` – Demonstrates defining and calling a simple function
- `function_with_arguments.c` – Demonstrates passing parameters to a function
- `function_with_return.c` – Demonstrates returning a value from a function
- `recursive_function.c` – Demonstrates recursion using factorial example
- `pass_by_values.c` – Demonstrates passing arguments by value (copy of variables)
- `pass_by_reference.c` – Demonstrates passing arguments by reference using pointers

## Key Concepts Covered
- Function declaration (prototype)
- Function definition
- Function calling
- Passing arguments
- Returning values
- Local variables
- Recursion
- Passing by value
- Passing by reference (using pointers)
