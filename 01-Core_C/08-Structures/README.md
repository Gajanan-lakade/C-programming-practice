# Structures in C Programming

This folder contains programs demonstrating the use of structures in C.
Structures allow grouping of different data types under a single name.

## Programs Included
- `basic_structure.c` – Demonstrates structure declaration and member access
- `array_of_structures.c` – Demonstrates storing multiple records using structures
- `structure_with_pointer.c` – Demonstrates accessing structure members using pointers

## Key Concepts Covered
- Structure declaration using `struct`
- Accessing structure members using dot (`.`) operator
- Array of structures
- Pointer to structure
- Arrow (`->`) operator
