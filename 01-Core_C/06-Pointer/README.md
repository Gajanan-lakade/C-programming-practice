# Pointers in C Programming

This folder contains basic programs demonstrating the concept of pointers in C.
A pointer is a variable that stores the memory address of another variable.

## Programs Included
- `basic_pointer.c` – Demonstrates pointer declaration, initialization, and dereferencing
- `pointer_with_array.c` – Demonstrates pointer usage with arrays and pointer arithmetic

## Key Concepts Covered
- What is a pointer
- Address-of operator (`&`)
- Dereference operator (`*`)
- Pointer declaration and initialization
- Printing memory addresses
- Pointer arithmetic
- Relationship between arrays and pointers
