# Arrays in C Programming

This folder contains basic programs demonstrating the use of arrays in C.

## Programs Included
- `basic_array.c` – Demonstrates array declaration, initialization, and element access
- `sum_array.c` – Calculates the sum of all elements in an array

## Key Concepts Covered
- Array declaration and initialization
- Indexing of array elements
- Looping through arrays using `for` loop
- Basic array operations (sum of elements)
